# 🌌 Aura Intelligence — AI Team Synthesizer

Aura is a high-fidelity, AI-driven single-page application designed to revolutionize how teams are formed. By leveraging advanced neural mapping and behavioral analytics, Aura synthesizes high-performance teams based on deep skill-matching and compatibility metrics.

![Aura Header](public/showcase/dashboard.png)

## ✨ Stellar Features

### 🚀 Cinematic Landing Experience
- **Ultra-Smooth Navigation**: Powered by **Lenis**, providing a luxurious, fluid scrolling experience.
- **Interactive Neural Mesh**: A dynamic background featuring glassmorphism spheres that repel the mouse cursor and are interconnected by a floating neural network grid.
- **Parallax Reveals**: Feature sections scale and slide into view using **Framer Motion**, creating a high-impact narrative flow.

### 🧠 Intelligence Mapping (Step 01)
- **Deep Skill Profiling**: Users can map their core expertise across multiple dimensions with granular mastery levels.
- **Aesthetic Entry**: A minimal, glassmorphism-based form designed for rapid-fire participant entry.

### 📊 Strategic Dashboard
- **2x2 Analytics Grid**: Real-time insights into total participants, unique skill counts, and average skill density.
- **AI Synthesis Engine**: One-click team generation powered by **Groq AI**, ensuring balanced composition and skill coverage.

### 📈 Deep Behavioral Analytics
- **Neural Radar Charts**: Visualize team dimensions (Performance, Stability, Synergy, Tech Depth, Innovation) using **Chart.js**.
- **Compatibility Scoring**: AI-generated reports explaining the "why" behind each team composition.

## 🛠️ Tech Stack

- **Core**: React 19 + Vite
- **Animations**: Framer Motion
- **Scrolling**: Local Lenis Smooth Scroll
- **Visualization**: Chart.js
- **Intelligence**: Groq AI (Llama 3 / Mixtral models)
- **Styling**: Vanilla CSS (Modern Design System with Glassmorphism)

## 🚀 Getting Started

### Prerequisites
- Node.js (v18+)
- npm or yarn

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/aura-team-builder.git
   cd aura-team-builder
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run the development server**
   ```bash
   npm run dev
   ```

4. **Build for production**
   ```bash
   npm run build
   ```

## 📂 Project Structure

- `src/pages/LandingPage`: The high-impact entry experience.
- `src/pages/SkillProfilePage`: Participant onboarding and expertise mapping.
- `src/pages/DashboardPage`: Real-time stats and AI generation controls.
- `src/pages/TeamsPage`: Visualization of synthesized teams.
- `src/pages/AnalyticsPage`: Deep-dive charts and AI compatibility reports.
- `src/services`: AI generation logic and API integrations.

## 🎨 Design Philosophy: "Neural Luminary"
Aura follows the **Neural Luminary** design system, characterized by:
- Deep obsidian surfaces (`#0d0d15`).
- Vibrant primary/secondary glows (Violet & Pink).
- High-quality blurs and glassmorphism.
- Micro-interactions that make the data feel alive.
